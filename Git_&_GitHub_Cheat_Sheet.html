Git & GitHub Cheat Sheet

This is a cheat sheet containing most important and commonly used Git commands.

What is Git?

  • Git is a free open-source distributed version control system that allows developers to keep track of their changes made to their coding files.
  • It also allows developers to collaborate with other developers and work along on a project.

What is Version Control System?

  • A version control system(VCS) is a system which records all the changes made to a file. So in future if any version of the file is required, it can be called.
  • VCS also ensures that all the team members are working on the latest version of the file.

What is GitHub?

  • GitHub is an open-source hosting platform where developers host their projects, store their codes, and so on.
  • It also allows other developers to collaborate on the projects.

Commands:-

  • git config --global user.name "firstName middleName lastName"
    to set a user name
  • git config --global user.email "valid emailID"
    to set a user valid emailID
  • git config --global user.name
    to check user name
  • git config --global user.email
    to check user valid emailID

  • pwd
    to know present working directory
  • cd "directoryName"
    to change directory
  • cd ..
    to go back to a previous directory
